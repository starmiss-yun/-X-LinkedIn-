# 马斯克在推特说了什么？利用 X 和 LinkedIn 打造你的全球化信息源

在互联网、金融、外贸或 Web3 行业，信息差就是金钱。如果你还在等国内媒体翻译的“二手新闻”，那你已经输了。要把这些平台当成你的“生产力工具”。https://pikayun.com/

1. 推特 (X)：构建你的行业情报网
不要把推特当微博刷。善用 List (列表) 功能。

建立不同的 List，比如“AI 大佬”、“Crypto 资讯”、“设计灵感”。

把相关的大V丢进去，这样你就得到了一个没有广告、纯度极高的信息流。你可以第一时间看到 OpenAI 发布了什么，而不是等两天后的公众号文章。

2. LinkedIn：职业社交礼仪
LinkedIn 是找外企工作、拓展海外客户的神器。

把你的 Profile 用英文精心打磨。

注意： LinkedIn 会显示“谁看了你的档案”，所以不要像视奸前任一样频繁点进别人的主页。

3. 节点技巧：为了账号，请用“固定IP”
对于职业账号，最大的风险是风控封号。

风险操作： 早上 9 点你的 IP 在美国，9 点半变成了日本，10 点变成了英国。推特的安全系统会认为你的号被盗了，直接冻结。

解决方案： 养成习惯，选定一个稳定的节点（最好是 IPLC 专线）后，就长期使用这一个。不要频繁切换地区。如果条件允许，使用“静态 IP”服务是最稳的。

4. 职场人的合规素养
公私分明： 如果是公司业务需要（如运营企业出海账号），请务必向公司申请合规的跨境专线，保留审批记录。不要用个人的机场去处理公司敏感业务。

商业保密： 在 LinkedIn 或 FB Group 交流时，严守商业机密。国外的商业间谍比你想象的要多，不要因为对方是“外国友人”就放松警惕。
